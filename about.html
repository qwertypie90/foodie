Team: Iris | Project name: .Wiki

In this project, we will create a unique wikipedia site for this class, so that anyone from the class can create an article for publication and share quick tips or tutorials about class topics, assignment, homework, etc. The main goal is to help students to find information faster and have a community where everyone can participate and help each other. Also, it can be served as a source bank that a student can always go back for a reference. The website function is simple and straightforward. The main landing page will display a list of brief articles such as HTML, CSS, and Javascript. Students can select any of these topics to read the entire the article.
